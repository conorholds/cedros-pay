# Cedros Pay

Unified payment infrastructure for Stripe and Solana x402.

## Packages

| Package | Description | Directory |
|---------|-------------|-----------|
| [`@cedros/pay-react`](ui/) | React web UI library | [`ui/`](ui/) |
| [`@cedros/pay-react-native`](react-native/) | React Native mobile library | [`react-native/`](react-native/) |
| Server | Rust backend API | [`server/`](server/) |

## Quick Start

### React Web
```bash
npm install @cedros/pay-react
```

### React Native
```bash
npm install @cedros/pay-react-native
```

See individual package READMEs for detailed documentation.
